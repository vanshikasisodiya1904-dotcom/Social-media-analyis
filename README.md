# Social Media Engagement Analysis

## Project Description
This project analyzes social media engagement data to understand how different content types and posting times affect performance.

##  Tools Used
- Python
- Pandas
- Matplotlib

##  Features
- Analysis of likes based on content type
- Identification of best time to post
- Visualization using graphs

##  Key Insights
- Reels generate the highest engagement
- Evening and Night posts perform best
- Video content outperforms images

## Files in this Project
- analysis.py → Python code
- social_media_data.csv → Dataset
